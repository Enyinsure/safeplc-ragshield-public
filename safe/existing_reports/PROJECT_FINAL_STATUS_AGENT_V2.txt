三、Agent v2 能力

Agent v2 在 Agent v1 + Safety Guard v1 基础上增加：

1. 多轮澄清判断
   对缺少型号、订货号、模块名称、电源类型的问题，先追问，不直接查库。

2. 自动追问
   根据缺失信息生成追问，例如：
   - 请补充具体 CPU / 模块型号或订货号
   - 请说明系统电源、负载电源、CPU 本体电源或 PS/PM 电源模块
   - 请给出被询问对象完整名称

3. 补充上下文后继续执行
   用户通过 --context 或 Streamlit 输入补充上下文后，Agent v2 自动生成增强后问题并调用 Agent v1。

4. 防误检索
   对“某个模块的电源电压允许范围是多少”等问题，在无上下文时停止自动检索，避免误命中其它模块。

5. 安全继承
   高风险问题不要求用户补充危险操作细节，直接交由 Safety Guard v1 完整拒答。

6. 通用问题不过度追问
   PROFINET / HMI / EMC 等具备通用证据的问题直接执行，不强制要求型号。

7. 交互式前端
   Streamlit Agent v2 支持：
   - 问题输入
   - 补充上下文输入
   - 强制检索开关
   - 澄清提示
   - 高风险提示
   - Agent v2 完整输出展示
