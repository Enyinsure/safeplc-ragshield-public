# Private Data Notice

This folder contains code and reports for SafePLC-SecGuard / SafePLC-Agent RAG runtime.

The following items are required only for local offline demonstration and should NOT be publicly redistributed on GitHub:

- Original Siemens manuals, such as:
  - 1500_manual_collection_zh-CHS_zh-CHS.pdf

- ChromaDB vector databases:
  - s7_chroma_db/
  - s7_chroma_db_v2/
  - s7_chroma_db_v2_stable/
  - s7_chroma_db_v3_tables/

- Full manual-derived outputs:
  - s7_output/
  - s7_output_test_chs_20/
  - s7_rag_processed/
  - s7_table_stitch_output*
  - mineru_ocr_test_*
  - multimodal extracted pages/images/chunks if they contain manual content

For public GitHub release, upload code, scripts, evaluation samples, risk labels, design documents, and reports only.
Do not upload the original Siemens manuals, full manual chunks, or derived vector databases.
